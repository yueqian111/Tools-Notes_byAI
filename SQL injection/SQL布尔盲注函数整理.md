|函数名|语法格式|功能说明|实战示例（Less-8场景）|
|---|---|---|---|

#### 函数组合逻辑（盲注核心）

布尔盲注的本质是**函数组合+逻辑判断**，比如逐位猜解数据库名的完整逻辑：

```SQL

# 猜解数据库名第n位的ASCII码是否等于x
1' and ascii(mid(database(),n,1))=x--+
```

- 若页面显示`Welcome Dhakkan` → 条件为真（x是第n位的ASCII码）

- 若页面无回显 → 条件为假（x不是目标值）
